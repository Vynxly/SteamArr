## üîë Getting Started
1. **Get a SteamGridDB API Key**
   - Click the "Get Key" link in the app
   - Sign in to SteamGridDB and copy your API key

2. **Set Your API Key**
   - Click "API Key" button
   - Paste your key and save

3. **Add Your Games**
   - Browse for your game executable
   - Search and select artwork
   - Add to Steam!
